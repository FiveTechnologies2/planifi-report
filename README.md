# Capítulo I: Introducción
## 1.1. Startup Profile
### 1.1.1. Descripción de la Startup
Five Technologies es una empresa creada por estudiantes de la Universidad Peruana de Ciencias Aplicadas, con el objetivo de desarrollar una aplicación que te brinde la capacidad de realizar las planillas a empresas "PYME" de una manera automática.  El nombre de nuestra aplicación es Planifi. Dentro de nuestra aplicación las empresas pyme pueden acceder a una suscripción dependiendo del personal de estas empresas para poder gestionar las plantillas de su personal, asimismo se brinda reportes financieros y una evaluación de horas empleadas en proyectos. Todo esto facilitará la gestión de dichas empresas porque disminuye su margen de error a la hora de realizar tanto pago como control de sus proyectos.
